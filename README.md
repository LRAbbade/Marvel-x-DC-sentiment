# SO_Threads
Trabalho de threads do renzão

Para executar, faça git clone, e então crie um arquivo "keys.py", onde deve se inserir as access keys do twitter de sua conta no seguinte formato:

```
consumer_key = ''
consumer_secret = ''
access_token = ''
access_token_secret = ''
```

Insira as keys entre ' '

Execute com

```
python main.py
```
